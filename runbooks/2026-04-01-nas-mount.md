## Task
Mount NAS at 192.168.50.101 (share: ServerBackup) to /mnt/nas for restic backups.

## Playbook Used
`playbooks/nas.yml`

```bash
ansible-playbook playbooks/nas.yml --ask-vault-pass
```

## What Changed
- Installed `cifs-utils` package
- Created mount point `/mnt/nas`
- Deployed CIFS credentials to `/etc/nas-credentials` (mode 0600, root-only)
- Added fstab entry for persistent CIFS mount with `_netdev,nofail` options
- Created `/mnt/nas/backups/homelab` subdirectory for restic

## Verification Steps
1. `df -h /mnt/nas` — should show the CIFS mount
2. `ls /mnt/nas/backups/homelab` — directory should exist
3. `cat /etc/fstab | grep nas` — fstab entry present
4. `sudo reboot` then verify mount comes back automatically

## Rollback
```bash
sudo umount /mnt/nas
sudo sed -i '/ServerBackup/d' /etc/fstab
sudo rm /etc/nas-credentials
sudo rm -rf /mnt/nas
sudo apt remove cifs-utils
```
